# union-health

协和益生公众号 h5 页面

## 本地开发 Vscode 配置

- Live Sass Compiler

将 `src/sass/` 目录下的 `Scss` 编译至 `src/css`目录

```js
"liveSassCompile.settings.formats": [
    // You can add more
    {
      "format": "expanded",
      "extensionName": ".css",
      "savePath": "/css"
    }
  ]
```
